#  Machine Learning Based Real Estate Price Prediction

##  Project Overview
This project aims to predict real estate property prices using Machine Learning techniques. 
The system analyzes property features such as area, location, number of bedrooms, and other relevant attributes to estimate the expected selling price.

A web-based interface is developed using Flask, where users can enter property details and get real-time price predictions.

---

##  Objective
To build a predictive model that accurately estimates property prices and compare multiple Machine Learning algorithms to identify the best-performing model.

---

##  Technologies Used
- Python
- Pandas & NumPy
- Scikit-Learn
- XGBoost
- Flask
- HTML & CSS

---

##  Machine Learning Models Implemented
- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

After comparison, XGBoost achieved better performance in terms of accuracy and error reduction.

---

## Project Workflow
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis
4. Model Training
5. Model Evaluation & Comparison
6. Model Deployment using Flask

---

##  Features
- Cleaned and preprocessed real estate dataset
- Model comparison and performance evaluation
- Web-based interface for user input
- Real-time price prediction
- Saved trained model using pickle (.pkl)
